🪙 KITCO Metals Widget

A modern WPF desktop widget that displays live gold and silver price charts using a clean dark UI.

📌 Overview

KITCO Metals Widget is a lightweight Windows desktop application built with WPF (.NET) that provides quick access to live gold and silver price charts. Designed as a compact, always‑available widget.
The app is optimized for minimal screen footprint and fast loading.

✨ Features

📈 Live Metals Charts

- Embedded WebView2 charts for Gold and Silver
- Auto‑refreshing data
- Clean tabbed interface with color‑coded accents

🎨 Modern UI

- Dark theme with gold/silver highlights
- Script‑style title font (Vladimir Script)
- Smooth tab transitions
- High‑contrast layout for readability
